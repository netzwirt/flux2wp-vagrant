# flux2wp-vagrant

Vagrant box which helps to import a fluxcms blog into a WP Installation.

## Description
Just run `vagrant up` to create a fresh WP install with pre-installed [fluxcms importer](https://github.com/netzwirt/wordpress-flux-importer)

## Access
- URL: http://10.3.45.3/wp-admin/admin.php?import=fluxcmsblog
- User: wp4
- PW: wp4

## Attention
`vagrant provision` deletes all data from previous import
